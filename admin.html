Dashboard Settings & Database Integration

Live Cloud Sync Integration Guide

By default, your dashboard is running in Local Storage Mode (all changes are saved inside your local browser only). To make your website live across the internet so anyone can see your updates in real-time, connect it to a free Firebase Cloud Database.

Step-by-step Setup instructions:

  1. Go to the Firebase Console.
  2. Click Add Project, type a project name (e.g. badrul-huda-academy), and create the project.
  3. In the project sidebar, click Build > Firestore Database, then click Create database.
  4. Choose a location close to your users, start in Test mode (so reads/writes are allowed), and click Create.
  5. Back on the Project Overview page, click the web icon (</>), register your app, and copy the firebaseConfig object.
  6. Open the project file main.js in a code editor and paste your credentials into the window.firebaseConfig block at the top.
  7. Save the file and refresh this page. The status badge will change to green: "Live (Firebase Connected)"!
